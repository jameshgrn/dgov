# Getting Started

Deterministic kernel for multi-agent orchestration via git worktrees.

## Install

```bash
uv pip install -e .
```

Requires Python 3.12+, git.

## Usage

```bash
dgov init                # Bootstrap .dgov/project.toml
dgov fix "Refactor utils" -f src/utils.py -f src/main.py  # One-off fix
dgov run plan.toml       # Execute a compiled plan
dgov status              # Show task status
dgov watch               # Stream events
dgov sentrux check       # Run architectural quality check
dgov clean               # Clean stale worktrees and outputs
dgov recover             # Recover from a crashed run
```

## Plan format

Plans are TOML files that compile to DAGs:

```toml
[plan]
name = "example"

[tasks.add-feature]
prompt = "Add the feature to src/foo.py"
agent = "kimi-k2.5"
files = ["src/foo.py"]

[tasks.add-tests]
prompt = "Write tests for the feature"
agent = "kimi-k2.5"
files = ["tests/test_foo.py"]
depends_on = ["add-feature"]
```
